## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or pnpm or yarn

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/sukunenv/picpic-cafe-web.git
   cd picpic-cafe-web
   ```
2. Install dependencies
   ```bash
   npm install
   ```

### Environment Setup
Create a `.env` file in the root directory:
```env
VITE_API_URL=http://localhost:8000/api/v1
```

### Running the App
Start the development server:
```bash
npm run dev
```
